escalc cheatsheet

so many estimators, so little time

Published

July 18, 2026

escalc contains a bewildering number of effect size estimators. We here provide some information on alternative estimators for the main effect size types so far covered in this tutorial. We conclude by recommending preferred estimators for other designs, including single group effect sizes (e.g., proportions). Note we do not discuss effect size estimators that are not recommended for use in ecology and evolution, such as raw correlation coefficients or unstandardized mean difference.

SMD and lnRoM options

I wrote this when i thought these alternative formulations were comparable with SMD. Now that I know they are not, I’m not sure about having this here…Might move to appendix

The standard estimators for SMD are “SMD” and “SMDH”, the latter of which is for heteroscedastic data. The standard estimators for lnRoM are “ROM”. There is no heteroscedastic estimator for lnRoM. This table includes alternative estimators provided by escalc and whether they are comparable to the standard estimator (either SMD, SMDH, or ROM). Note that paired design estimators require an additional input ri, an estimate of correlation between groups and take a single sample size (ni) instead of a sample size for two groups (n1i, n2i)

class escalc Heteroscedasticity Paired designs Comparable to standard? Details
SMD “SMCC” No Yes No Paired group formulation that uses the standard deviation of group 1 and 2.
“SMCR” No Yes No Paired group formulation that uses standard deviation of group 1 only.
“SMCRH” Yes Yes No Same as above but for heteroscedastic data.
“SMCRP” No Yes No An alternative formulation that uses the standard deviations of both group 1 and 2.
lnRoM “ROMC” No Yes Yes Paired formula for lnRoM.

Note that “Zmu” is only available in our own function eff_this (originally published in Nakagawa et al. XXXX) and supplied here. To select a paired formula, set paired_design = TRUE and be sure to provide r and n as an input.

Look into this: “See also Morris and DeShon (2002) for a thorough discussion of the difference between the”SMCC” and “SMCR” change score measures”

Not sure when you should use SMCC versus SMCRP I don’t think any of them are compatible with SMD unpaired…Gahhh

lnOR / lnRR options

The standard estimator options in escalc are “OR” for lnOR and “RR”. The alternative estimators below are for “matched pairs contingency tables”.

class escalc Paired designs Comparable to standard? Details
lnOR “MPOR” yes? ? “marginal log odds ratio”
“MPORC” yes? ? “conditional log odds ratio”
“MRPETO” yes? ? “conditional log odds ratio estimated with Peto’s method
lnRR “MPRR” yes? ? “marginal log-risk ratio”
“MPRD” yes? ? “marginal risk difference”

Zr options

You can also calculate point biserial correlation coefficient “RPB” and biserial correlation coefficient “RBIS” but these return an estimate of ‘r’ and should not be analyzed directly in a meta-analytic model

class escalc Comparable to standard? Details
Zr “ZBP” Yes “point biserial correlation coefficient directly to Zr”
“ZBIS” Yes “biserial correlation coefficient directly to Zr”
